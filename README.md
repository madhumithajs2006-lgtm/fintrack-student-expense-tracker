# 💰 FinTrack - Student Expense Tracker

FinTrack is a responsive expense tracker dashboard built using HTML, CSS and JavaScript.

It helps students manage their income, expenses and savings efficiently.

## 🚀 Features

- 💳 Add income and expenses
- 📊 Expense analytics chart
- 🔍 Search transactions
- 💾 Local storage support
- 🗑️ Delete transactions
- 📄 Export transactions to CSV
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
